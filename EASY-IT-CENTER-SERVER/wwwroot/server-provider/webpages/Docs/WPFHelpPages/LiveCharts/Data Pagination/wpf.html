Data Pagination

You can always use Axis.MinValue and Axis.MaxValue properties to set the range of an axis, in this example we have a line series with 100 points, and we are only showing the 25 records the user wants to see.

To keep this example always up to date it is directly pulled from the Github repository, the repo for simplicity uses the UserControl class to wrap every example, but you can use any container for your plots.
