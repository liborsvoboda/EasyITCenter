Installation

This article is built using visual studio 2015
Offline install? please see: extract binaries from nuget

Install using the package manager console

PM> Install-Package LiveCharts.Wpf

Or go to Solution Explorer, right click on references, then Manage NuGet Packages…

Browse for LiveCharts.Wpf, select the package and click on install

Add namespace to your XAML

xmlns:lvc="clr-namespace:LiveCharts.Wpf;assembly=LiveCharts.Wpf"

And to code behind if necessary

using LiveCharts;
using LiveCharts.Wpf;
