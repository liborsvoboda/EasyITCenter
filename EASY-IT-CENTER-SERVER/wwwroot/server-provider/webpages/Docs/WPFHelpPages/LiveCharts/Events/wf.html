Events

To keep this example always up to date it is directly pulled from the Github repository, the repo for simplicity uses the UserControl class to wrap every example, but you can use any container for your plots.

Events available in the library

Name Description
Chart.UpdaterTick Fired every time The chart updates
Chart.DataClick Occurs every time the users click over a point in a chart
Chart.DataHover Happens every time the users hovers over a point in a chart
Axis.RangeChanged Occurs every time the range of an axis changes, including when a user uses features such as zooming and panning

Drag a CartesianChart object from your toolbox, you should be able to add the desired event using the properties tool box, as you do with any other control.
