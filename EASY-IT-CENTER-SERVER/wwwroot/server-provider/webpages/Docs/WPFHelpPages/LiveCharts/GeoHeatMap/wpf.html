GeoMaps

The GeoMap control is really easy to customize, here is a collection of maps around the world, you can also create your own.

GeoMaps are alpha, and they should have more features, create a new issue in the github repo with your ideas.

To keep this example always up to date it is directly pulled from the Github repository, the repo for simplicity uses the UserControl class to wrap every example, but you can use any container for your plots.
