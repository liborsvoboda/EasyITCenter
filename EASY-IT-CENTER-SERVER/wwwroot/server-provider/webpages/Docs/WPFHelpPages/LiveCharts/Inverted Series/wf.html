Inverted Series

Inverting series in LiveCharts is practically automatic, you must use the inverted version of the series, then live charts will handle the inverted draw, notice the configuration for Vertical and Horizontal series is different, and you must conciser this if plotting a custom type.

Practically all series have a built in inverted version, for example the LineSeries class inverted version is VerticalLineSeries class, in The Cartesian chart article you could find a full list with the supported series.
