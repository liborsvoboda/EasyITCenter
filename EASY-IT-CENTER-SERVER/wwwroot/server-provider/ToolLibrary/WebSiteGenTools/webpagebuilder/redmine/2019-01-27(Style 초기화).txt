 2) 선택 후 Delete 또는 Copy시 Style 설정창이 보이는 버그
 (수정완료)
    1. Delete 시 보이지 않도록 수정완료
    2. Copy 시 보이지 않도록 수정완료

 3) 선택 후 Move Block시 Style 설정창이 보이는 버그
