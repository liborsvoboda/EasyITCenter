### Compress a file/folder
```sh
tar -zcvf <zip_file_name>.tar.gz /path/to/folder/or/file
```

### Uncompress a file/folder
```sh
tar xvf <zip_file_name>.tar.gz /optional/folder/path/to/keep/unzip/content
```
